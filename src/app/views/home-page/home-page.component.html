Αν είσαι μεταξύ 18-35 ετών, σε ενδιαφέρει να αποκτήσεις γνώσεις γύρω από το Web Application Development και να εργαστείς ως Full-stack web developer (front-end και back-end), φτιάξαμε για σένα ένα ξεχωριστό, δωρεάν πρόγραμμα εκπαίδευσης με όλα τα web application trends!

Από 04/12 έως 22/01 Bizrupt (Βικέλα 34-36, Ηράκλειο)

Σχετικά με το πρόγραμμα

Το πρόγραμμα απευθύνεται σε όσους και όσες κατέχουν βασικές γνώσεις προγραμματισμού και επιθυμούν να αναπτύξουν τις γνώσεις τους πάνω στα Web Application. Έχει δημιουργηθεί σε συνεργασία με τον Γιάννη Μακρίδη, Co-Founder & CTO της Tourmie. Μέσα από την εκπαίδευση θα μάθεις πώς να δημιουργείς τα δικά σου responsive Web Application, θα εξοικειωθείς με το Object-Oriented Programming (OOP), θα μάθεις να γράφεις κώδικα JavaScript/TypeScript και θα γνωρίσεις διαφορετικά back-end environments.

Το πρόγραμμα θα διαρκέσει από τις 04 Δεκεμβρίου 2023 έως και τις 22 Ιανουαρίου 2024. Αποτελείται από 33 ώρες δια ζώσης θεωρητικής και πρακτικής εκπαίδευσης, με 2 τρίωρα μαθήματα την εβδομάδα (11 συνολικά), κάθε Δευτέρα και Τετάρτη και ώρες 18:00 και 21:00. Φυσικά, θα κάνουμε διάλειμμα για τις γιορτές των Χριστουγέννων. Οι θέσεις είναι μόνο 25 και γι’ αυτό θα γίνει επιλογή μεταξύ των αιτήσεων που θα λάβουμε.

Πληροφορίες συμμετοχής

Σε αφορά, αν ενδιαφέρεσαι:
  • Να λάβεις τις βασικές γνώσεις γύρω από το Web Application development.
  • Να εργαστείς ως Full-stack web developer (front-end και back-end).
  • Να μάθεις πώς λειτουργεί το Web, το Web development stack και οι πιο δημοφιλείς Web development τεχνολογίες.
  • Να μάθεις να δημιουργείς απλά responsive Web Application, χρησιμοποιώντας Angular, Bootstrap, HTML και CSS.
  • Να μάθεις να γράφεις κώδικα με asynchronous JavaScript/TypeScript.
  • Να μάθεις να γράφεις σε back-end environments, χρησιμοποιώντας Node.js και να τα συνδέεις με databases (για data storage/retrieval).
  • Να εξοικειωθείς με τον Αντικειμενοστραφή Προγραμματισμό (Object-Oriented Programming (OOP) paradigm).
  • Να επιλέγεις τις κατάλληλες τεχνολογίες για το κάθε Project.
  • Nα αναπτύξεις τα skills που απαιτούνται για να ξεκινήσεις να εργάζεσαι, μέσα από τη δημιουργία του δικού σου Project.
Τι ψάχνουμε να δούμε σ’ εσένα:
  • Θέληση για γνώση!
  • Όρεξη για Full-stack web development (front-end και back-end).
  • Συνέπεια στις ώρες των μαθημάτων.
  • Επιθυμητό: σπουδές ή εμπειρία στο αντικείμενο του προγραμματισμού.
  • Να έχεις μαζί σου laptop
